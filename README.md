①課題名
つなぎノート〜交換日記アプリ〜

②課題内容（どんな作品か）
カウンセリングサービスの提供を想定して作成した、カウンセラーとクライアントをつなぐ交換日記Webアプリです。

ユーザーはカレンダーから日付を選び、その日の出来事や気持ちを日記として記録できます。
カウンセラーは内容を確認してコメントを返すことができ、セッション外でも日々の状況を見守ることができます。
データはlocalStorageに保存されるため、ページを閉じても記録が残ります。

⑤工夫した点・こだわった点
【デザイン】
全体のカラートーンを統一し、縦のラインを揃えることで、視覚的にすっきりとした見た目になるよう意識しました。
ヘッダーにはGeminiで生成したオリジナルの犬のイラストを取り入れ、温かみのある雰囲気を演出しました。
【カレンダー】
当日の日付はフォントを大きく表示し、ひと目で今日がわかるように工夫しました。
また日記を記入した日には花丸マークを表示することで、記録できた達成感を感じてもらえるようにしました。
【カウンセラー欄の配慮】
カウンセラーのコメント欄はユーザーが編集できないようreadonly属性を設定し、背景色を暗くすることで視覚的にも
「入力不可」とわかるよう配慮しました。誤操作を防ぎつつ、役割の違いが伝わるUIを意識しました。
